# External Alpha Runs

This directory is reserved for empirical external alpha run records.

Use it for live external runs only.
Do not place curated benchmark packs, synthetic pressure fixtures, or local estimate-only comparison inputs here.

Recommended per-run layout:

- `REPORT.md`
- `.synrail/` when available
- preserved local verification outputs that matter for review

Use [docs/review/EXTERNAL_ALPHA_RUN_TEMPLATE_001.md](../../docs/review/EXTERNAL_ALPHA_RUN_TEMPLATE_001.md) for each run report.
Use [docs/review/EXTERNAL_ALPHA_LEDGER_2026-05.md](../../docs/review/EXTERNAL_ALPHA_LEDGER_2026-05.md) as the summary ledger for the next ten runs.

Current benchmark and comparison fixtures elsewhere in `fixtures/` are still useful, but they are not empirical external measurements.
